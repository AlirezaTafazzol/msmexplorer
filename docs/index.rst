MSMExplorer is a Python visualization library for statistical models of
biomolecular dynamics. It provides a high-level interface for drawing
attractive statistical graphics with `MSMBuilder <http://msmbuilder.org>`_.

To get started with MSMExplorer, please refer to the
:ref:`installation page <installing>`. You may also want to browse the
:ref:`example gallery <example_gallery>` to get a sense for what you can do
with MSMExplorer and then check out the :ref:`notebooks <notebooks>` and
:ref:`API reference <api_ref>` to find out how.

<https://github.com/msmexplorer/msmexplorer>`_.
