Examples
========

The following examples show off various aspects or capabilities of
MSMExplorer. They can be run interactively in Jupyter (IPython) notebook.
Download the `notebook files
<https://github.com/MSMExplorer/MSMExplorer/tree/master/examples>`_ and open
them in Jupyter::

    $ jupyter notebook

.. To make the ipython rendered images show up, each rst file must be
   in its own directory.
